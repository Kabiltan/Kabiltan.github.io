Pong

Description

This project is a web application that allows users to play pong with two player and one player options. My motivation for this project was a desire to build a classic game from scratch. I chose pong because it was one of the first games to ever be played on a computer and it wasn't too complicated to recreate.

Development

Although the HTML and CSS were important, the backbone of this project was the Javascript. I used Javascript to code everything from the background and score to the ball and paddles. This project was especially fun to work on because I was able to incorporate my knowledge of physcis to program the ball movement.

For this project, I wrote the main menu/user interface in HTML and CSS with a little bit of Javascript. The actual game itself was coded in Javascript.
